# Trivia Game Scorer

Create a `TriviaScorer` class with a `sumScore(answers : string) : int` 
 
It will take an empty or one character `answers`. `“Y”` indicates a correct question and returns `1`. `“X”` indicates an incorrect answer and returns `0`.

Examples:
`sumScore(“”)   // 0`
`sumScore(“X”)  // 0`
`sumScore(“Y”)  // 1`
