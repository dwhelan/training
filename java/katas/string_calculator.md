# String Calculator

Create a `StringCalculator` class with a `add(input : string) : int` method.

It should allow 0, 1 or 2 numbers, separated with commas in `input` and should return the sum of those numbers.

Some examples:

```
game = new StringCalculator();

game.add("");    // 0
game.add("1");   // 1
game.add("1,2"); // 3
```
